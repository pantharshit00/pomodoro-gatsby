## Pomodoro Clock

Pomodoro timer that works offline. https://pompom.now.sh

## Getting started
1. Clone this repo.
2. Installed dependencies
  ```
  # prefer yarn as this has the lockfile
  npm install
  # OR
  yarn 
  ```
3. Run gatsby
  ```
  npm run develop
  # OR
  yarn run develop
  ```
